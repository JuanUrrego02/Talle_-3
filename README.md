# 🧠 E-commerce Web Application
La aplicación web de comercio electrónico es una plataforma integral de compras en línea creada con React, React Router y Material UI. Ofrece una experiencia de usuario fluida, que permite a los clientes explorar productos, añadir artículos a su carrito y realizar el pago de forma segura. La aplicación está diseñada para ser escalable, fácil de mantener y fácil de usar.

## 🚀 Features
* Interfaz fácil de usar con diseño adaptativo
* Catálogo de productos con funciones de filtrado y clasificación
* Carrito de la compra con funciones para añadir, eliminar y actualizar cantidades
* Proceso de pago seguro con integración de pasarela de pago
* Función de productos favoritos con almacenamiento local persistente
* Diseño adaptativo para dispositivos móviles y tabletas

## 🛠️ Tech Stack
* Frontend: React, React Router, Material UI
* Backend: None (client-side only)
* Database: Local storage (para el carrito y los productos favoritos)
* Build Tool: Vite

## 📦 Installation
Para instalar la aplicación, siga estos pasos:
1. Clone el repositorio utilizando `git clone`.
2. Navegue hasta el directorio del proyecto utilizando `cd`.
3. Instale las dependencias utilizando `npm install` o `yarn install`.
4. Inicie el servidor de desarrollo utilizando `npm run dev` o `yarn dev`

## 💻 Usage
Para utilizar la aplicación, siga estos pasos:
1. Abra un navegador web y vaya a `http://localhost:3000`.
2. Explore los productos y añada artículos a su carrito.
3. Vea su carrito y actualice las cantidades según sea necesario.
4. Realice el pago de forma segura utilizando la pasarela de pago.
